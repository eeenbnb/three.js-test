# webpack + typescript + three
```
npm init
npm i --save webpack webpack-dev-server typescript awesome-typescript-loader source-map-loader
npm i --save three @types/three
```

## webpack setting
edit ``webpack.config.js``

## typescript setting

edit ``tsconfig.ts``

## exec
```
npm start
```

## build
```
npm build
```
